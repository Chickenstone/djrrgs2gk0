# 腾讯云静态网站托管部署指南

既然我们已经在使用腾讯云开发 (CloudBase) 的数据库服务，那么最完美、最原生的国内部署方案就是：**腾讯云开发 - 静态网站托管**。

它不仅在国内访问速度极快（自带腾讯云 CDN），而且与我们现在的数据库环境是同一个体系，管理起来非常方便。

## 部署步骤

### 1. 构建前端生产代码
在终端运行以下命令，将我们的 React 代码打包成最终的静态文件：
```bash
npm run build
```
执行完毕后，你的项目根目录下会生成一个 `dist` 文件夹。

### 2. 在腾讯云开启“静态网站托管”
1. 登录 [腾讯云开发控制台](https://console.cloud.tencent.com/tcb)。
2. 进入你之前创建的环境。
3. 在左侧菜单栏找到 **“静态网站托管”**。
4. 点击 **“开通”**（如果是首次使用，需要几分钟的初始化时间）。

### 3. 上传文件
开通成功后，你会看到一个文件管理的界面。
1. 点击 **“上传文件”**（或直接拖拽）。
2. 选择你本地电脑上 `dxapp/dist` 文件夹里面的**所有文件和文件夹**上传上去。（注意：是上传 `dist` 里面的内容，而不是上传 `dist` 这个文件夹本身）。

### 4. 获取访问域名
上传完成后，在“静态网站托管”页面的上方，你会看到一个 **“基础配置” -> “默认域名”**（类似 `xxxx.tcloudbaseapp.com`）。
点击这个域名，你就可以在国内流畅地访问你的“东兴便民通”应用了！

---

## 高级选项（使用命令行自动化部署）

如果你不想每次都手动拖拽上传，可以使用腾讯云开发的命令行工具 `@cloudbase/cli`：

1. 全局安装 CLI：
```bash
npm i -g @cloudbase/cli
```

2. 登录腾讯云：
```bash
tcb login
```

3. 一键部署 `dist` 目录（将 ENV_ID 换成你的真实环境ID）：
```bash
tcb hosting deploy ./dist -e YOUR_ENV_ID
```
