# 东兴便民通 - 项目维护与修改指南

如果您需要对“东兴便民通”小程序的内容进行修改或日常维护，请参考以下指南。本项目采用了**前端页面+云端数据库**的架构，因此修改内容分为两类：**修改数据**和**修改界面**。

---

## 1. 如何修改“动态数据”？
*(如：美食列表、景点列表、文创商品)*

这些数据已经接入了**腾讯云开发 (CloudBase)**。如果您需要增加、删除或修改这些内容，**完全不需要修改代码**。

**操作步骤：**
1. 登录 [腾讯云开发控制台](https://console.cloud.tencent.com/tcb)。
2. 进入您的环境，点击左侧菜单的 **“数据库”**。
3. 找到对应的集合 (Collection)：
   - 要改美食：进入 `restaurants` 集合。
   - 要改景点：进入 `spots` 集合。
   - 要改文创商品：进入 `products` 集合。
4. 在记录列表中，点击对应的记录进行“编辑”，或者点击“添加记录”新增一条。修改保存后，小程序端会自动同步最新数据。

---

## 2. 如何修改“静态界面与文案”？
*(如：首页的轮播图、政务大厅的办事指南、各个按钮的文字)*

这些内容是写死在前端代码里的。如果您需要修改，请打开代码编辑器（如 Trae / VS Code），找到对应的文件进行修改。

### 📍 核心页面修改定位表

#### 🏠 首页 (`src/pages/home/index.tsx`)
- **想换轮播图图片**：搜索代码中的 `CAROUSEL_IMAGES` 数组，替换里面的 `url` 链接。
- **想改“为您推荐”的内容**：搜索 `RECOMMENDATIONS` 数组，修改 `title`、`desc` 或 `image`。

#### 🏛️ 政务模块 (`src/pages/government/index.tsx`)
- **想改办事指南的列表**：搜索 `SERVICES` 数组，修改 `title`（标题）或 `desc`（描述）。

#### 🛠️ 一条龙服务 (`src/pages/service/index.tsx`)
- **想改服务包的价格和内容**：搜索 `PACKAGES` 数组，修改里面的 `title`、`price`、`features` 等字段。

#### 👤 个人中心 (`src/pages/user/index.tsx`)
- **想改底部的版权声明**：拉到代码最底部，找到 `@chickenstone` 相关的文本直接修改。

## 3. 如何修改“样式与颜色”？

本项目使用 **Tailwind CSS** 来编写样式。如果您想改变某个按钮或背景的颜色，只需在对应的 `.tsx` 文件中修改元素的 `className`。

*   **蓝色系**：`bg-blue-500`, `text-blue-600`
*   **橙色系**：`bg-orange-500`, `text-orange-500` (美食模块常用)
*   **绿色系**：`bg-green-500` (攻略模块常用)
*   **红色系**：`bg-red-600` (文创模块常用)

例如，把一个蓝色按钮改成红色：
```tsx
// 修改前
<button className="bg-blue-600 text-white">确定</button>

// 修改后
<button className="bg-red-600 text-white">确定</button>
```

---

## 4. 修改代码后如何发布上线？

如果您修改了任何 **代码文件（.tsx）**，需要重新打包并上传到腾讯云，用户才能看到更新。

1. 在终端运行打包命令：
   ```bash
   npm run build
   ```
2. 登录腾讯云控制台 -> 静态网站托管。
3. 将项目目录下新生成的 `dist` 文件夹内的**所有内容**，拖拽上传并覆盖旧文件。
4. （可选）在静态网站托管的“缓存配置”中点击“刷新缓存”，确保用户立即看到最新版本。
